# DrawLots 프로젝트
- 수업용 예제 프로그램
- DrawLots는 학급에서 발표 순서를 랜덤하게 결정해주는 파이썬 GUI 프로그램입니다.

소스코드명 : drawlots_main.py


## 시작하기

이 프로그램을 사용하기 위해서는 Python이 시스템에 설치되어 있어야 합니다. Python의 설치 방법은 [Python 공식 웹사이트](https://www.python.org/)를 참고하세요.
